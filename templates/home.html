ABOUT

Easily find the prerequisites of the courses you're planning on taking with a click of a button. The website displays all the engineering courses in the clicked engineering type, upon clicking on any of the courses shown, it's prerequisites will light up, as well as the official description of the course will appear on the right hand side.

The creation of this site was inspired by another website, created by PresidentKevvol. Both sites serve the same purpose, however their implementations, limitations, and visualizations differ.

This website is still under maintenance, and so it would be wise to double check the prerequisites with an advisor or from the official course description website
